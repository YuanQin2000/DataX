size_t  len 
)

Calculate the max common matched length of the prefix and postfix sub string.

This max length will determinate that how long step the pattern string should be move forward. This is the key concept of the KMP algorithm.

Here is the key point of the max common matched length. 1) P1 P2 P3, ..., P(k-1), P(k), ..., P(j-k), P(j-k+1), ..., P(j)

Parameters
pStringThe pattern string.
pArrayThe buffer to store the max common matched length data.
lenThe length of the string and the array.
